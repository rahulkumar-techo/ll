🚀 LL (Language Learning)

LL is an AI-powered language learning platform focused on real-world communication and practical speaking skills. Instead of traditional memorization, LL helps users learn through interactive scenarios, adaptive quizzes, and personalized feedback.

✨ Features
🧠 AI-based personalized learning
🗣️ Scenario-based conversation practice
🎯 Adaptive difficulty system
📊 Progress tracking & analytics
📱 Mobile-first (built with React Native / Expo)
💡 Vision

To make language learning simple, practical, and accessible, especially for users who want to build confidence in real-life conversations.

🛠️ Tech Stack
Frontend: React Native (Expo), NativeWind
Backend: Node.js, Express
Database: MongoDB
AI: OpenAI / LLM-based services
📌 Use Cases
Interview preparation
Daily conversation practice
Travel communication
Skill improvement for students
🔥 Getting Started
# install dependencies
pnpm install

# start expo app
pnpm start
